## 1. 목적
- 토비의 스프링 학습
- 깃허브의 버전 관리 기능 학습

## 2. self 활용 규칙
- TDD를 적용합니다
- 초난감 DAO 부터 출발합니다
- 한가지 주제의 리팩토링이 끝난후 테스트 케이스를 모두 통과하면 커밋합니다
- 커밋 할 때에는 각 리팩토링의 내용을 간략하게 정리합니다
